# QQ音乐静态项目HTML源码

## 资源描述

本仓库提供了一个QQ音乐静态项目的HTML源码，该源码展示了一个简单的音乐推荐页面。页面包含以下几个标签：

- **为你推荐**
- **背景音乐**
- **情歌**
- **网络歌曲**
- **官方歌单**
- **经典**
